Remote File Move · Privacy Policy

Privacy Policy

Last updated: May 29, 2026

Remote File Move is a Chrome extension companion for a user-controlled PC-to-PC file bridge. It checks bridge status and can start a browser download from a bridge endpoint that the user has configured.

Data collection

Remote File Move does not collect, sell, or transfer user data to the publisher or to third parties. The extension does not require an account and does not include analytics or advertising code.

Local settings

The extension may store the bridge URL, current PC side, and optional bridge token in Chrome local storage on the user's device. These settings are used only to connect to the user's configured bridge.

File transfer data

When the user uses the download helper, files are requested from the user's configured bridge endpoint and saved through Chrome downloads. The publisher does not receive those files, file names, tokens, or bridge activity.

Permissions

  • storage: Save local bridge settings.
  • downloads: Save a pending bridge ZIP when the user chooses to download it.
  • notifications: Show local completion/status notifications.
  • alarms: Periodically refresh bridge status while Chrome is running.
  • host permissions: Connect only to the configured trusted bridge/localhost endpoints included in the extension package.

Remote code

The extension does not load or execute remote JavaScript or WebAssembly. All extension code is included in the package submitted to the Chrome Web Store.
