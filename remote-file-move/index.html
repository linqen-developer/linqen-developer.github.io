Remote File Move

Move files between two Chrome browsers.

Install the extension on both Chrome browsers, create a pairing code on the sender, enter it on the receiver, and download the received files.

How it works

  1. Open Remote File Move and choose Send files.
  2. Select files and share the pairing code.
  3. Open Remote File Move on the receiving Chrome and choose Receive files.
  4. Enter the code and keep both tabs open until transfer completes.

No native helper

This Chrome-to-Chrome version does not install a native app and does not access the Windows Explorer file clipboard. It moves files inside the browser using WebRTC.

Privacy

Pairing uses a random ntfy.sh signaling room. File contents are sent over the WebRTC data channel between browsers when the connection succeeds.
