Remote File Move

Install the native helper for real file movement.

Remote File Move uses a Chrome extension plus a native Windows helper. Chrome alone cannot set the Windows FileDrop clipboard, so the helper is required for real Ctrl+C → Ctrl+V file movement between trusted PCs.

Install

  1. Extract the ZIP on each trusted Windows PC.
  2. Run install-helper.cmd.
  3. Install/open the Remote File Move Chrome extension.
  4. Open the popup and confirm the helper status is ready.

This helper build requires Node.js 20+ on the Windows PC.

Privacy

Files are transferred only through your configured trusted bridge. The publisher does not receive your files, file names, tokens, or bridge activity.
