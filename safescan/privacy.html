SafeScan by Linqen Privacy Policy

Last updated: May 15, 2026

SafeScan by Linqen is a simple scanner app for QR codes, one-page paper capture, PDF picking, and sharing scan results.

Information SafeScan collects

SafeScan does not collect personal information, account information, payment information, contact lists, advertising identifiers, or location data.

SafeScan does not require an account or login.

Camera permission

SafeScan requests camera permission only when the user chooses to scan a QR/barcode or capture a paper page.

Camera input is used on the device for scanning or creating the selected result. SafeScan does not send camera images or scan content to a Linqen server.

Files and PDFs

When the user chooses an existing PDF, SafeScan uses the Android system file picker. Selected files are handled locally so the user can view or share the chosen result.

SafeScan does not upload selected PDFs or generated files to a Linqen server.

Local history

SafeScan may keep recent scan results visible inside the app during use so the user can return to them quickly. This information stays on the user's device.

Users can clear local app data through Android settings or by uninstalling the app.

Sharing and selling data

SafeScan does not sell user data.

SafeScan does not share user data with third parties for advertising, analytics, profiling, creditworthiness, or unrelated purposes.

If the user taps a share action, Android's native share sheet may let the user send the selected result to another app. That action is controlled by the user and the receiving app's own policies.

Children

SafeScan is a general utility app and is not directed to children. It does not knowingly collect information from children.

Changes

If this policy changes, the updated version will be posted on this page with a new date.
