AI Prompt Capsule Privacy Policy

Last updated: May 23, 2026

AI Prompt Capsule is a local-first Chrome extension that lets users save reusable prompt context, organize it into folders, and paste the selected capsule or folder into a focused browser text field.

Information AI Prompt Capsule handles

AI Prompt Capsule may handle the following information only to provide its core prompt-capsule feature:

  • Prompt capsule titles, folders, body text, and tags created by the user.
  • Text selected by the user when the user chooses to clip selected page text into a capsule.
  • The currently focused browser text field, only so the selected capsule text can be inserted where the user asks.

AI Prompt Capsule does not ask for a name, email address, account, payment information, passwords, or login credentials.

Local-first storage

Prompt capsules are stored in Chrome extension storage on the user's own browser profile using chrome.storage.local.

AI Prompt Capsule does not send prompt capsules, selected text, page content, URLs, or usage activity to a Linqen server.

Page access

AI Prompt Capsule runs a content script on webpages so the user can open the capsule palette and insert saved prompt context into the text field they are using.

The extension does not automatically collect browsing history, monitor keystrokes, or upload webpage content. Text is inserted only when the user chooses a capsule or folder. Selected text is saved only when the user explicitly chooses the clip/save action.

Analytics, ads, and remote code

AI Prompt Capsule does not use analytics, advertising SDKs, tracking pixels, or behavioral profiling.

AI Prompt Capsule does not use remote JavaScript or remote WebAssembly. Its extension code is included in the submitted Chrome Web Store package.

Sharing and selling data

AI Prompt Capsule does not sell user data.

AI Prompt Capsule does not transfer user data to third parties except as required by the Chrome browser platform to run extension features requested by the user.

AI Prompt Capsule does not use user data for advertising, creditworthiness, lending decisions, or purposes unrelated to its single purpose.

User control and deletion

Users can edit or delete saved capsules inside the extension. Users can also remove all local extension data by clearing AI Prompt Capsule's extension storage in Chrome or uninstalling the extension.

Changes

If this policy changes, the updated version will be posted on this page with a new date.
