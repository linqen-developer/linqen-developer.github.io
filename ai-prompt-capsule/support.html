AI Prompt Capsule Support

Last updated: May 23, 2026

AI Prompt Capsule helps you save reusable AI prompt context locally, organize it into folders, and paste the right capsule or folder into the browser text field you are using.

Quick start

  1. Open an AI chat, email, document, form, or another page with a text field.
  2. Click the text field where you want prompt context to be inserted.
  3. Press Ctrl + Shift + K on Windows/Linux or Command + Shift + K on macOS.
  4. Choose a folder and click All, or open a folder and click Paste on one capsule.

Managing capsules

Use the extension popup to create, edit, delete, search, and organize capsules. Capsules are stored locally in your Chrome browser profile.

Troubleshooting

  • If paste does not work, click the target text field again and retry the shortcut.
  • If a site blocks direct text insertion, paste into a normal text field first, then copy it into the target app.
  • If your capsules disappeared after switching Chrome profiles, check that you are using the same browser profile where they were saved.
  • To reset the extension, remove AI Prompt Capsule from Chrome or clear the extension's local storage.

Privacy

AI Prompt Capsule has no account, analytics, backend, or network sync. See the privacy policy for details.
