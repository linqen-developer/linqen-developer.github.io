table export Privacy Policy

Last updated: May 28, 2026

table export is a Chrome extension that lets users export a visible HTML table from a webpage as a CSV file.

Information table export handles

The extension can detect visible HTML tables on webpages where it runs. When the user clicks the Export CSV button, it reads the text content of the selected table cells to generate a CSV file.

table export does not ask for a name, email address, account, payment information, passwords, login credentials, or browsing-history export.

How information is used

Selected table content is used only to create a local CSV file for the user to download. The extension does not use table content for analytics, advertising, profiling, or unrelated purposes.

Local-first processing

CSV generation is performed locally in the browser. table export does not upload webpage table content, page data, or generated CSV files to a server controlled by table export or Linqen.

Storage

table export does not store exported table content in a remote database. Temporary browser memory and the downloaded CSV file may exist only as part of the user-triggered export flow.

Sharing and selling data

table export does not sell user data.

table export does not transfer user data to third parties except as required by the Chrome browser platform to run extension code and create a user-triggered download.

table export does not use user data for advertising, creditworthiness, lending decisions, or purposes unrelated to its single purpose.

Remote code

table export does not use remote JavaScript or remote WebAssembly. Its extension code is included in the submitted Chrome Web Store package.

User control and deletion

Users choose whether to export a table by clicking the Export CSV button. Closing the webpage, clearing browser data, deleting downloaded CSV files, or uninstalling the extension removes extension-side temporary state.
