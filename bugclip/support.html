BugClip Support

Help for BugClip, a no-login bug packet Chrome extension for web teams.

Contact

Email support: contact@linqen.co.kr

Quick start

  1. Open the webpage where the issue is visible.
  2. Click the BugClip extension icon.
  3. Choose Capture current screen.
  4. Mark the problem with a pin, box, arrow, text label, or blur box.
  5. Write a short description.
  6. Choose Download ZIP · recommended to send the developer a complete local bug packet.

What the ZIP contains

  • screenshot.png — annotated screenshot.
  • metadata.json — URL, page title, browser context, annotations, and error/click signals when available.
  • report.md — developer-readable Markdown report.

Pages BugClip cannot capture

Chrome blocks extensions from capturing some browser-managed pages, including chrome:// pages, Chrome Web Store pages, extension pages, browser settings, and DevTools pages. Open a normal website tab and try again.

Privacy reminder

BugClip suggests blur regions for common sensitive fields, but screenshots can still contain visible private information. Review the preview and add blur boxes before sharing a bug packet.
