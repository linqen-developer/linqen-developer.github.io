photo diet Privacy Policy

Last updated: May 28, 2026

photo diet is a Chrome extension that compresses image files locally in the user's browser.

Information photo diet handles

photo diet handles image files only when the user chooses or drops them into the extension popup. The extension may read file names, file sizes, image dimensions, and image pixel data solely to create compressed output files.

photo diet does not ask for a name, email address, account, payment information, passwords, browsing history, or login credentials.

How information is used

Chosen image files are used only to generate compressed JPG, PNG, or WebP output and package the results into a downloadable ZIP file.

Local-first processing

Compression is performed locally in the browser using built-in browser image and canvas features. photo diet does not upload image files, image metadata, or compressed outputs to a server controlled by photo diet or Linqen.

Storage

photo diet does not store image files in a remote database. Temporary browser memory and downloadable files may exist only as part of the user-triggered compression and download flow.

Sharing and selling data

photo diet does not sell user data.

photo diet does not transfer user data to third parties except as required by the Chrome browser platform to process files selected by the user.

photo diet does not use user data for advertising, creditworthiness, lending decisions, or purposes unrelated to its single purpose.

Remote code

photo diet does not use remote JavaScript or remote WebAssembly. Its extension code is included in the submitted Chrome Web Store package.

User control and deletion

Users control which files are opened in photo diet. Closing the popup, clearing browser data, or uninstalling the extension removes extension-side temporary state.
