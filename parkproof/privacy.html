Park Proof by Linqen Privacy Policy

Last updated: May 21, 2026

Park Proof by Linqen is a simple local parking-note app for saving a parking clue, meter/return time, optional photo name, and shareable return note.

Information Park Proof collects

Park Proof does not collect personal information, account information, payment information, contact lists, advertising identifiers, precise location data, or background location data for Linqen.

Park Proof does not require an account or login.

Parking notes

When the user creates a parking note, the information entered by the user may include a parking clue, garage level, row or pillar marker, meter/zone context, timer minutes, return note, and optional photo file name.

These parking notes are stored locally on the user's device so the user can view or share them later. Park Proof does not upload parking notes to a Linqen server.

Photos and files

If the user chooses to attach a photo, Park Proof uses the Android system picker. File/photo access happens only after the user chooses that action.

Park Proof does not upload selected photos or file names to a Linqen server.

Location and tracking

Park Proof does not automatically track location, does not run background GPS tracking, and does not create a route history.

The app is designed for manual parking notes entered by the user.

Local history and deletion

Park Proof keeps recent parking notes in a local app log on the user's device. Users can clear the local log inside the app.

Users can also remove local app data by clearing app storage through Android settings or by uninstalling the app.

Sharing and selling data

Park Proof does not sell user data.

Park Proof does not share user data with third parties for advertising, analytics, profiling, creditworthiness, or unrelated purposes.

If the user taps a share action, Android's native share sheet may let the user send a parking note to another app. That action is controlled by the user and the receiving app's own policies.

Ads, analytics, and servers

Park Proof does not include ads SDKs or analytics SDKs in the current version.

Park Proof does not require a Linqen app server for the current local parking-note flow.

Children

Park Proof is a general utility app and is not directed to children. It does not knowingly collect information from children.

Changes

If this policy changes, the updated version will be posted on this page with a new date.
